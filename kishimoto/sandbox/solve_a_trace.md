# solve_a 往復ロボット 動作トレース

## オートマトン定義（2状態, 初期方向=R の場合）

| 状態 | 壁なし(c=0) | 壁あり(c=1) |
|------|-------------|-------------|
| 0(前進) | F → 状態0 | R → 状態1 |
| 1(反転) | R → 状態0 | R → 状態0 |

## トレース: 横セグメント (i,0)〜(i,2), 初期 (i,0) R 状態0

| step | pos   | dir | state | wall? | action | 結果              |
|------|-------|-----|-------|-------|--------|-------------------|
| 1    | (i,0) | R   | 0     | no    | F→s0   | → (i,1) R s0     |
| 2    | (i,1) | R   | 0     | no    | F→s0   | → (i,2) R s0     |
| 3    | (i,2) | R   | 0     | yes   | R→s1   | → (i,2) D s1     |
| 4    | (i,2) | D   | 1     | *     | R→s0   | → (i,2) L s0     |
| 5    | (i,2) | L   | 0     | no    | F→s0   | → (i,1) L s0     |
| 6    | (i,1) | L   | 0     | no    | F→s0   | → (i,0) L s0     |
| 7    | (i,0) | L   | 0     | yes   | R→s1   | → (i,0) U s1     |
| 8    | (i,0) | U   | 1     | *     | R→s0   | → (i,0) R s0     |

→ step 8 の結果 = step 1 の開始状態 → **周期 = 8, 過渡期なし**
→ パトロールセル = {(i,0), (i,1), (i,2)} ✓

## サイクル長

セグメント長 L の場合: サイクル長 = 2L + 2

## 縦セグメント (DIR_D) の場合

同じオートマトン。DIR_D で開始 → 下に直進 → 壁で R(=左向き) → R(=上向き) → 上に直進 → ...
→ 縦方向に往復。

## 最適化: 二部グラフ最小頂点被覆

- 左ノード = 横セグメント, 右ノード = 縦セグメント
- 辺 = セルを共有する (H-seg, V-seg) ペア
- 各セルは exactly 1つの H-seg と 1つの V-seg に属する
- 最小頂点被覆 = 全セルを被覆するセグメントの最小集合
- König の定理: 二部グラフでは min vertex cover = max matching → 多項式時間で最適解

## コスト
- 長さ1セグメント → 1状態 (spinner)
- 長さ2+セグメント → 2状態 (oscillator)
- 壁は追加しない (A_W=1000)
